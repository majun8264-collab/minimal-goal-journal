# 极简目标日记部署说明

## 适合的方式

推荐先部署到 GitHub Pages。部署后手机可以用固定网址打开，不需要和电脑在同一个网络。

## 注意隐私

这个版本已经移除了内置日记照片和旧日记示例。你每天填写的内容会保存在手机浏览器本地，不会自动上传到云端。

如果你把年度目标也视为隐私，请把 GitHub 仓库设为私有，或后续改用带登录的云同步方案。

## GitHub Pages 部署步骤

1. 在 GitHub 新建一个仓库，比如 `minimal-goal-journal`。
2. 把本文件夹里的内容上传到仓库根目录。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 1-3 分钟。
7. 打开 GitHub 给出的 Pages 地址。

## 手机上变成 App

1. 用 Safari 打开 GitHub Pages 地址。
2. 点分享按钮。
3. 选择“添加到主屏幕”。
4. 名称填写“极简目标日记”。

## 数据备份

当前版本的数据保存在当前浏览器本地。建议每周点一次“导出记录”，把 JSON 文件保存到 iCloud、微信文件或电脑里。
